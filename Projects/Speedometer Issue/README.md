# Speedometer Issue - Santana PS10 (24-BX-HS)

## Symptom

- Shows 10% slow after 80-100 km of driving
- Recovers after 15 minutes cooling
- Needle wanders at constant speed

## Likely Cause

Heat-related speed sensor or connector issue on gearbox/transfer case.

## To Check

- Sensor connector corrosion
- Wiring near exhaust (heat damage)
- Wiggle test on connector when acting up

## Dependency

Must be fixed before cruise control retrofit. The EDC needs a reliable speed signal for cruise control to work.
