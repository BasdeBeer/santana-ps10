# Santana PS10 Body Electrical System

**Source:** Service Manual PS10, Section 8 (Pages 8-1 through 8-4)

---

## General Description

The electrical components of the Santana PS10 are designed to work at 12 volts from the electric battery system, with the negative terminal as ground or earth.

---

## Wiring Color Codes

To ease identification of each circuit, a colour code is used for wiring throughout the vehicle:

| Wire Color Code | Color |
|---|---|
| GY | Gray |
| WG | White-Green |
| Br | Brown |
| L | Blue |
| B | Black (ground) |
| R | Red (power) |

## System Voltage and Ground Distribution

- **Voltage**: 13.5V nominal (12V charging system)
- **Ground**: Negative return through engine block and chassis
- **Main Ground Distribution**:
  - Engine block to chassis
  - ECU/Electrical components to engine block
  - Component return paths via dedicated grounds

## Connector Standards

**Typical Vehicle Connectors:**
- Multiple-connector design with color-coded wires for quick identification
- Sealed connectors for environmental protection
- Redundant ground paths for reliability
- All connectors standardized across vehicle systems

## Electrical System Components Overview

### Power Supply Circuit
- 13.5V battery system
- Generator for battery charging
- Multiple relay and fuse distribution

### Lighting System
- Head lamps (left and right)
- Side lights and turn signals
- Rear lights and license light
- Interior lighting

### Control Systems
- Starting motor control
- Wiper motor control
- Heating system control
- Brake light activation

### Safety Features
- Fused circuits for overcurrent protection
- Ground distribution for equipment safety
- Sealed connectors to prevent moisture ingress

---

## Notes for Service

- All electrical work must be performed with the battery disconnected
- Use proper color-coded wires when making repairs
- Ensure all connections are clean and secure
- Sealed connectors must be protected from moisture
- Reference the complete electrical diagrams in Section 10 for detailed component specifications

*For complete wiring diagrams, electrical specifications, and troubleshooting procedures, refer to Section 10: Technical Data - Electrical Diagrams (Pages 10-5 through 10-19)*
