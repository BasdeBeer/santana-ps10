# Transmission System Overview - Section 7A

## Manual Transmission (ZF LT-85)

The Santana PS10 is equipped with a ZF LT-85 five-speed manual transmission, completely synchronized with one reverse gear. This section covers the transmission, clutch, transfer case, and differential systems.

### General Description

**Transmission Type:** ZF LT-85 Five-Speed Manual
**Synchronization:** Completely synchronized forward speeds with friction cones and ball retainers
**Reverse Gear:** Straight-cut
**Transmission Code:** LT-85

The transmission features five forward gears with optimized ratios for engine efficiency. All gears corresponding to forward speeds are permanent and helicidal construction. The reverse gear is straight-cut design.

The synchronised couplings are grooved by electro-erosion, POCKET-type, to help avoid slipping in harsh working conditions.

### Gear Ratios and Speed Relations

#### First Gear Ratio
- **Value:** 29/21 or 29/21

#### First Speed Ratio
- **Value:** 1.380

#### Gear Ratios by Speed

| Speed | Gear Ratio | Speed Ratio | Total Speed Reduction |
|-------|-----------|------------|----------------------|
| Reverse | - | - | - |
| 1st | 36/13 or 37/14 | 2.768 | 3.820 |
| 2nd | 30/19 or 26/25 | 2.643 | 3.647 |
| 3rd | - | 1.519 | 2.179 |
| 4th | - | 1.040 | 1.435 |
| 5th | 19/33 | 0.576 | 0.795 |

### Transmission Specifications

**Manual Transmission Type:** LT-85 ZF Five-Speed
**First Gear Ratio:** 29/21 (driven/drive gears)
**First Speed Ratio:** 1.380
**Synchronisation:** Fully synchronized (all forward speeds)
**Reverse Gear:** Straight-cut design

### Key Technical Characteristics

- Helicidal gears for forward speeds provide smooth, quiet operation
- Synchronised engagement prevents grinding during gear changes
- POCKET-type synchroniser grooves prevent slipping under high load
- Reverse gear is straight-cut for robust engagement
- All gears optimized for progressive speed steps allowing optimum engine use

### Components and Assembly

The transmission assembly consists of:
- Primary shaft with input gears
- Main shaft with output gears
- Counter shaft (auxiliary shaft)
- Reverse shaft assembly
- Selector forks and synchroniser mechanisms
- Clutch housing interface
