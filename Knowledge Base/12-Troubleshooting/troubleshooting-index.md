# Troubleshooting Index

This section serves as a cross-reference for common issues. When diagnosing a problem, check the relevant sections below for procedures and part numbers.

## Engine Issues
- **Won't start / hard start:** Check fuel system (02-Engine/Fuel-and-Emissions/), EDC codes (08-Electrical/EDC-Diagnostics/), glow plugs
- **Overheating:** See cooling system (02-Engine/Cooling-System/)
- **Low power / turbo issues:** Check fuel and emissions (02-Engine/Fuel-and-Emissions/), exhaust (02-Engine/Exhaust/)
- **Oil pressure warning:** See lubrication (02-Engine/Lubrication/)
- **EDC warning light:** Read blink codes (08-Electrical/EDC-Diagnostics/edc-blink-codes.md)
- **Timing belt:** Check interval at 90,000 km (01-Maintenance-and-Lubrication/)

## Transmission Issues
- **Gear shifting problems:** ZF gearbox (03-Transmission/ZF-Gearbox/)
- **Transfer case noise/selection:** Transfer case (03-Transmission/Transfer-Case/)
- **Differential noise/leaks:** Differentials (03-Transmission/Differentials/)
- **Clutch slipping/heavy pedal:** Clutch (03-Transmission/Clutch/)

## Brakes
- **Spongy pedal / poor braking:** Brakes service (05-Brakes/), check fluid level and condition
- **Brake disc replacement:** Rear disc procedure (05-Brakes/rear-brake-disc-replacement.md)
- **Handbrake not holding:** Parking brake adjustment (05-Brakes/)

## Steering and Suspension
- **Wandering / pulling:** Front alignment (04-Steering-Suspension-Wheels/)
- **Steering heavy / noisy pump:** Servo-steering (04-Steering-Suspension-Wheels/Steering/)
- **Suspension noise:** Check silent blocks, dampers (04-Steering-Suspension-Wheels/)

## Electrical
- **Warning lights meaning:** Owner's guide instruments (11-Owner-Guide/instruments-controls.md)
- **Wiring issues:** Wiring diagrams (08-Electrical/Wiring-Diagrams/)
- **Lighting problems:** See electrical section (08-Electrical/Lighting/)

## Body
- **Heater not working:** Heater/ventilation (07-Body/Heater-Ventilation/)
- **Water leaks:** Body service (07-Body/)
- **Door/lock issues:** Owner's guide (11-Owner-Guide/Before-Driving/)

- Start with the figure index to find the right exploded diagram
