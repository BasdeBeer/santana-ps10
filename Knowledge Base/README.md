# Santana PS10 Knowledge Base

Knowledge database for a 2006 Santana PS10 station wagon with Iveco 8140.43P engine, ZF gearbox, ~120,000 km.

## How to Use

Ask Claude questions about your vehicle and it will search through this knowledge base to find answers. Examples:
- "What oil does my PS10 need?"
- "How do I replace the rear brake discs?"
- "What's the torque spec for the hub nuts?"
- "Find the part number for the fuel filter"
- "What does the EDC warning light mean?"
- "What maintenance is due at 120,000 km?"

Page numbers are referenced so you can look up the original PDF diagrams when needed.
