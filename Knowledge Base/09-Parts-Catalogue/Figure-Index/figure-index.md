## Additional Engine Notes

- **Type 1:** Mechanical injection system
- **Type 2:** Common rail diesel system with high-pressure pump
- Multiple variants per component based on engine configuration
- Shims available in precise increments for timing/valve clearance adjustment
- Bearing sets with standard, -0.254mm, and -0.508mm clearances for main and connecting rod bearings

## Parts Number Format Guide

**Santana Parts:** PS-XXXXXX
- Example: PS-791519 (Emblem "PS-10" RR)

**OEM Supplier Parts:** Alphanumeric with standard notation
- Example: A840815-040 (Cylinder block assy Type 1)
- Example: A60017-014 (Culasse assy complete Type 1)

**Cross-reference notation in remarks:**
- TYPE 1 / TYPE 2: Engine variant specification
- D:..., T:...: Dimensional specifications (Diameter, Thickness)
- AR: Adjustment required (commonly for shimming)
- STD: Standard specification
- EXC.: Excluding/Exclusive variant
