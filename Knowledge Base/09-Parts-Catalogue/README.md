## How to Use This Knowledge Base

### Finding a Part

**Method 1: If you know the figure number**
1. Go to [Figure-Index](Figure-Index/figure-index.md)
2. Find the figure and note which section it's in
3. Go to that section's file (e.g., Engine-Parts)
4. Locate the part in that figure's table

**Method 2: If you know the component name**
1. Choose the appropriate system file based on the component type
2. Search (Ctrl+F) for the component name
3. Find the corresponding figure and part number

**Method 3: If you have a part number**
1. Use the Parts Number Index (Section 7) in the original PDF
2. Or search across all markdown files for the part number

### Understanding the Part Tables

Each part is listed with these columns:
- **Ref:** Reference item number on the exploded diagram
- **Part No.:** Santana (PS-) or OEM supplier part number
- **Description:** Component name and function
- **Qty:** Quantity required per vehicle
- **Remarks:** Configuration notes (Type 1/2, optional, dimensional specs)

### Special Notations

- **TYPE 1 / TYPE 2:** Engine variant specific
- **D:** Diameter measurement (mm)
- **T:** Thickness measurement (mm)
- **M:** Metric screw size notation
- **EXC.:** Excluding/exclusive configuration
- **N/:** No/without configuration
- **W/:** With configuration
- **AR:** Adjustment required
