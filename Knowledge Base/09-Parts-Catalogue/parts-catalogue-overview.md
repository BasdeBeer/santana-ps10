# Santana PS-10 Parts Catalogue Overview

## Document Information

**Model:** Santana PS-10 (Type 1 / Type 2)
**Edition:** 1st Revised Edition (Updated Parts Info)
**Catalogue Number:** 791124I (1st REM) / 790960-02I (2nd edition)
**Published:** October 2004
**Manufacturer:** SANTANA MOTOR S.A., Linares, Spain

## Scope

This parts catalogue covers all service parts for the Santana PS-10 vehicle. It includes parts for two variants:
- **Type 1:** Mechanical Injection
- **Type 2:** Common Rail (Diesel)

## How to Use This Catalogue

The catalogue is organized using a two-step methodology:

### Step 1: Locate Figure in Illustration Index
- Updated figures are shown on pages 8-10
- If a part is not in the updated section, refer to the original version figures (pages 52-66)

### Step 2: Find Part Number
- Once you locate the figure/plate with the exploded diagram, identify the item number
- Cross-reference the item number in the parts list for that figure
- Use the Part Number Index (Section 7) for alternative lookup if needed

### Step 3: Consider Remarks
- Check the Remarks column for special notes (e.g., EXC., TYPE 1, TYPE 2, N/ AUX HEATER, W/ FR; RR AIR CON)
- These annotations indicate variants or specific configurations

## Part Number Format

Parts numbers in this catalogue follow these patterns:
- **Santana parts:** PS-XXXXXX (e.g., PS-791519)
- **OEM supplier parts:** Alphanumeric codes (e.g., A610067-002, 02162-05123)

## Remarks Notation

Common abbreviations in the Remarks column:
- **EXC.:** Exclusive / Excluding
- **TYPE 1/TYPE 2:** Specific variant
- **N/ AUX HEATER:** No Auxiliary Heater configuration
- **W/ FR; RR AIR CON:** With Front and Rear Air Conditioning
- **M6:** Metric screw size
- **QTY (Qty):** Quantity per vehicle

## Modification Notices

Parts modification notices are issued periodically. When changes occur, the catalogue indicates:
- **→ XXXXX-XXXXX (Part No):** New part number that is interchangeable with former part
- **← XXXXX-XXXXX (Part No):** Former part number being replaced
