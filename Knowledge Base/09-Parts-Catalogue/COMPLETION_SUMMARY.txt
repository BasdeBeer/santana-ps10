NOTES FOR FUTURE MAINTAINERS
================================================================================

1. PART NUMBER FORMAT
   - Santana parts: PS-XXXXXX (e.g., PS-791519)
   - OEM parts: Alphanumeric (e.g., A840815-040)
   - Always preserve exact format

2. VARIANT NOTATION
   - TYPE 1: Mechanical injection
   - TYPE 2: Common rail with electronics
   - Keep consistency across sections

3. MEASUREMENT NOTATION
   - D = Diameter (in mm)
   - T = Thickness (in mm)
   - M = Metric fastener (M6, M8, etc.)
   - Always include units

4. CONFIGURATION NOTES
   - EXC. = Excluding/exclusive variant
   - N/ = Without configuration
   - W/ = With configuration
   - LH/RH = Left/Right

5. SEARCH OPTIMIZATION
   - Use consistent terminology
   - Include synonyms in descriptions
   - Cross-reference frequently accessed items
