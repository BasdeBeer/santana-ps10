## Part Number System

### Santana Numbering
- **PS-XXXXXX**: Vehicle parts (e.g., PS-601039 = Rear lamp assembly)
- **791XXX**: Fluids and lubricants (e.g., 791003 = Engine oil SAE 10W-40)
- **790XXX**: Special tools and sealants (e.g., 790972 = Oil filter removal tool)

### Iveco/OEM Numbering
- **AXXXXXX-XXX**: Iveco engine and drivetrain components
- **Bosch XXXXXXX**: Electrical components and connectors

### Type Variants
- **TYPE 1**: Mechanical injection engine (older)
- **TYPE 2**: Common Rail diesel with EDC (Euro III)
