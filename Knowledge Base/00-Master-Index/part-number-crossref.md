## Notes & Conventions

1. **Part Number Prefixes:**
   - `PS-XXXXXX`: Santana Motor part numbers
   - `A-XXXXXX`: Iveco original equipment manufacturer numbers
   - `790XXX` / `791XXX`: Santana reference numbers (fluids, tools, sealants)

2. **Type Variants:**
   - **TYPE 1**: Mechanical Injection engine (earlier variant)
   - **TYPE 2**: Common Rail Diesel engine (Euro III, EDC control)

3. **Missing Data:** Parts marked with "-" indicate that specific OEM cross-reference numbers are not yet extracted from the original PDF manuals. The Knowledge Base contains placeholder entries awaiting detailed extraction.

4. **Measurements & Specifications:**
   - Thread/Bolt sizes shown as M6, M8, M10 (metric)
   - Wire gauge shown in AWG and mm²
   - Shim thickness (T:) shown in millimeters

5. **Cross-Reference Method:**
   - Start with the Santana PS- number to identify the part
   - Use the Iveco A-series number for OEM sourcing
   - Use reference numbers (791XXX/790XXX) for fluids and tools
