# Santana PS10 Knowledge Base

Documentation and knowledge base for a 2006 Santana PS10 station wagon.

## Vehicle

| | |
|---|---|
| **Engine** | Iveco 8140.43P, 2.8 Turbo Diesel, Euro III |
| **Turbo** | Mitsubishi TD04 |
| **Gearbox** | ZF LT-85 manual |
| **Transfer** | Part-time 4WD |
| **Odometer** | ~120,000 km |

## What's in here

- **`/Manuals/`** -- Original PDF service manual, parts catalogue, owner's manual, and training manual. Source of truth for all specs and procedures.
- **`/Knowledge Base/`** -- Extracted markdown summaries organized by vehicle system, with a precomputed index layer in `00-Master-Index/` for fast lookups.
- **`/Projects/`** -- Active retrofit and modification projects (cruise control, etc.).
- **`/Verzekering/`** -- Insurance documents.
- **`CLAUDE.md`** -- Instructions for Claude Code on how to use this repo.

## Using with Claude Code

Clone this repo and open it in Claude Code. The `CLAUDE.md` file tells Claude how to navigate the knowledge base, when to check source PDFs, and how to handle specs and part numbers. Ask it anything about the vehicle.
